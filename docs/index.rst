Introduction
############

Spotify API gives applications access to Spotify’s vast resources such as retrieving an artist’s tracks, getting track recommendations and managing playlists. 

Prerequisites
=============

* Create a free or premium Spotify account
* Create an account on the Spotify developer dashboard
* Get access tokens to make API requests
